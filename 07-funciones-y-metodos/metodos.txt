- El metodo 'toString', sirve para convertir a string un dato. 
- El metodo 'toUpperCase', sirve para convertir texto a mayuscula.
- El metodo 'toLowerCase', sirve para convertir textoa minuscula.
- El metodo 'length', sirve para calcular la longitud de un string (tambien cuenta los espacios).
- Para concatenar dos textos basta con unirlos con el operador '+'. Pero tambien existe la posibilidad de utilizar un metodo llamado 'concat', al que se le pasara como parametro lo que se quiera concatenar.
- El metodo 'indexOf', sirve para buscar una posición dentro de un string o array o lo que fuere. Este me va a arrojar un numero con la posicion del caracter inicial de la palabra o caracter que le pase, o el indice en caso que sea un array. Ademas de este metodo, se puede utilizar el metodo 'search' y sirve para lo mismo.
- El metodo 'lastIndexOf', sirve para lo mismo que 'indexOf', con la diferencia que este se utiliza en caso de que la busqueda de la palabra o caracter este repetida, y se necesite saber la posicion de la ultima coincidencia.
- El metodo 'match', sirve para hacer una busqueda de una palabra. Este metodo me devolvera un array con la posicion en la que se encuetra dicha palabra o caracter y el texto original y completo donde fue encontrado. Si a esta busqueda la encierro asi: '/busqueda/gi', buscara todas las palabras iguales a esa. 
- El metodo 'substr', sirve para sacar una palabra de un string, pasandole como primer parametro a partir de que caracter y como segundo parametro, cuantos caracteres tomar.
- El metodo 'charAt', sirve para sacar un caracter del string, pasandole como parametro la posicion que se desea saber.
- El metodo 'startsWith', sirve para buscar una palabra/caracter al comienzo del string. Si esa palabra/caracter se encuentra al inicio, el metodo devolvera un 'true', de lo contrario sera 'false'.
- El metodo 'endsWith', sirve para lo mismo que el 'startsWith', solo que este sera para verificar al final del string.}
- El metodo 'includes', sirve para buscar una palabra/caracter, en cualquier posicion del string y devolverá un booleano.
- El metodo 'replace', sirve para reemplazar una palabra dentro de un string. Se le pasara como primer parametro la palabra a reemplazar y como segundo el reemplazo.
- El metodo 'slice', sirve para buscar una posicion dentro de un string y captar lo que este a partir de dicha posicion. Tambien puedo pasarle más de un parametro para que tome un pedazo del string y no desde donde se le indica hasta el final.
- El metodo 'split', sirve para meter en un array todo lo que se le pase como parametro, sea el string entero o parte del string, pasandole una palabra/caracter.
- El metodo 'trim', sirve para recortar los espacios que tenga mi string, por delante o por atras.
- El metodo 'append', agrega el elemento de parámetro dentro de la etiqueta del elemento selector al final. Ejemplo en 'json.js'
- El metodo 'prepend', agrega el elemento de parámetro dentro de la etiqueta del elemento selector al principio. Ejemplo en 'json.js'
- El metodo 'document.createElement', sirve para crear una etiqueta nueva desde el codigo. Ejemplo en 'json.js'
