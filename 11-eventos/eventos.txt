- Para crear se debe utilizar el metodo 'addEventListener' y a este pasarle como parametro "entre comillas", el evento a realizar. Y seguido puede ir una función normal o funcion flecha, llamando a la función principal del cambio a realizar con el evento.
- Eventos de raton: 
    - click: click con el raton.
    - mouseover: pasar por encima con el raton.
    - mouseout: es lo contrario al evento 'over', cuando sale del foco del objeto el raton.
- Eventos de teclado: 
    - focus: evento que capta cuando se esta dentro de un campo de escritura.
    - blur: evento que capta cuando se sale del campo de escritura.
    - keydown: evento que capta cuando se esta [pulsando] una tecla. 
    - keypress: evento que capta cuando se [pulso] una tecla.
    - keyup: evento que capta cuando se suelta una tecla.
- Evento load:
    Este evento sirve para cargar apenas se abre el navegador con apuntando a la propiedad window. Es decir, si antes los scripts los ubicaba abajo del body, para que primero cargen los elementos del DOM y despues ejecutar, creando una funcion con este evento y encerrando dentro de ella todo el codigo JS, no sera necesario ubicar los scripts en la zona baja y se podra colocar dentro del head del HTML.
